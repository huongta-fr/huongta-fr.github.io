Generalizable instance segmentation under visual style and domain shifts

Duration: 6 months • Team scaled: 2

Building early prototypes and datasets for comic/manga character segmentation. : This internship aims to prepare the foundations for a broader research project on style- and domain-general instance segmentation, with applications to visual storytelling books such as manga, comics, webtoons, and illustrated books.

Key responsibilities & achievements

  • Benchmarking baseline instance segmentation methods: Mask2Former, SAM, BlindNet
  • Experiment with style-robust features, simple style normalization and augmentation techniques (e.g., MixStyle, AdaIN, RandAugment)
  • Collect, generate, prepare datasets with multiple styles.
  • Generate synthetic stylized data using existing models (e.g., Stable Diffusion, StyleGAN3) to simulate style variability.
